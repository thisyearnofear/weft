# Weft Scripts

## Deploy Scripts

### deploy.sh
Deploys the WeftMilestoneStaking contract to 0G Chain testnet.

```bash
./deploy.sh --network testnet
```

### setup.sh
Sets up ENS domains and configures initial project.

```bash
./setup.sh --project my-project
```

## Usage

```bash
cd scripts
chmod +x *.sh
./deploy.sh
```
